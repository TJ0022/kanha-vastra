# Kanha Vastra — Simple WhatsApp Storefront

A clean, single-page website for selling Laddu Gopal vastra, mukut, and accessories with WhatsApp ordering.

## Quick Start
1. Unzip this folder.
2. Open `index.html` in your browser to preview.
3. Replace images in `assets/` with your real photos (use the same filenames).
4. Edit phone number in two places:
   - In `index.html` (search `919265550647` and `9265550647`).
   - In `script.js` (`PHONE_INTL` constant). For India keep `919265550647`.
   - If your WhatsApp is not Indian, change to your country code (e.g., +971 → `971...`).

## Publish on GitHub Pages
1. Create a new repo on GitHub named `kanha-vastra`.
2. Upload all files from this folder to the repo root.
3. In GitHub > Settings > Pages:
   - Source: `Deploy from branch`
   - Branch: `main` / root
4. Your site will be live at `https://YOUR-USERNAME.github.io/kanha-vastra/`.

## Customize Text (Gujarati/Hindi/English)
- You can freely edit headings and paragraphs in `index.html`.
- Fonts are from Google Fonts: Inter + Playfair Display.

## SEO Tips
- Replace `og-cover.jpg` with a nice banner (1200×630).
- Edit `<title>` and `<meta name="description">` in `index.html`.
